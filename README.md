# Weezer — MySpoti

Sitio web creado como parte de la **Actividad(MySpoti)**.  
El proyecto presenta una pequeña web dedicada a la banda **Weezer**, con páginas sobre sus álbumes *The Blue Album*, *Pinkerton* y *Songs from the Black Hole*.  
Incluye información básica, un formulario de suscripción y reproductores de audio HTML para canciones como **"Buddy Holly"**, **"The Good Life"** y **"Blast Off!"**, todo desarrollado con **HTML5 semántico** y sin CSS, siguiendo buenas prácticas de estructura y validación.
